## How to add a new component (Codex instructions)

When a new .tsx file arrives in this folder:
1. Open lib/mdxComponents.ts
2. Add: import { ComponentName } from '@/components/visualisations/ComponentName'
3. Add to the mdxComponents object: ComponentName,
4. Run tsc --noEmit
5. Update the table above with the new component
6. Verify it renders at the chapter route that uses it
