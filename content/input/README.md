# Content Input

This folder stores raw chapter material before it is turned into app-ready content.

Examples:
- rough notes
- source summaries
- chapter outlines
- draft prompts for Codex
- reference snippets

Suggested convention:
- one folder per chapter
- keep raw text and source notes together

Example:
- `server-nic/notes.md`
- `server-nic/sources.md`
